# Channel Specifications

Required fields for each asset type. Every asset file must include all **Required** fields for its channel. **Optional** fields should be included when relevant to the campaign.

## Writing by Channel

Each channel has different rhetorical demands. Writing the same voice across all channels is a failure mode — an Instagram caption and a LinkedIn post serve different audiences in different mindsets. Adapt the voice to the channel while staying grounded in the organization's brand.

**Social (Instagram, Facebook, TikTok, Threads):** Conversational, first-person plural ("we") or direct address ("you"). Short sentences. The first line must earn the scroll-stop — lead with the most surprising, useful, or emotionally resonant element. No corporate throat-clearing. Hashtags are discovery tools, not decoration — use ones the audience actually searches.

**LinkedIn:** Professional but not stiff. The audience is in work mode — lead with an insight, result, or question that's relevant to their professional context. First ~140 characters appear before "see more," so the hook must land in that window. Avoid the LinkedIn cliché of starting with "I'm proud to announce" or "Thrilled to share."

**Email:** Subject line is the entire pitch for opening — make it specific, not clever. Preview text extends the subject, not repeats it. Body copy should be scannable: short paragraphs, one idea per paragraph, CTA visible without scrolling. Write as a person, not a brand — emails from "someone" outperform emails from "something."

**YouTube (long-form and Shorts):** Titles must work as search terms and click triggers simultaneously. Descriptions front-load key information in the first 2-3 lines (visible before "Show more"). Tags are for search — include what someone would actually type to find this content. Thumbnail text should be readable at thumbnail size (3-5 words max).

**Google Ads:** Headlines are constrained to 30 characters — every word earns its place. Descriptions (90 characters) should state the value proposition and CTA, not describe the brand. Keywords should reflect how the audience searches, not how the organization talks about itself.

**Website:** SEO titles and meta descriptions are for search results, not for the page visitor — write them for the person deciding whether to click. Body content should be structured for scanning: clear headings, short paragraphs, visual hierarchy. The first paragraph should tell the reader what they'll get from the page.

## Email

**Required:**
- **Subject Line**: The email subject
- **Preview Text**: Text shown in inbox preview (max ~90 characters)
- **Body Content**: Complete email copy with paragraph breaks
- **Call to Action**: Primary CTA text and destination URL

**Optional:**
- **Header Image**: Specifications (dimensions, subject, text overlay)
- **Segmentation Notes**: Which list segment(s) receive this email
- **Send Time**: Specific send time if different from publish date

## Instagram Post

**Required:**
- **Caption**: Full post text (up to 2,200 characters)
- **Image Requirements**: Format (Square 1080x1080, Portrait 1080x1350, or Landscape 1080x566), subject description, mood/style direction
- **Alt Text**: Descriptive accessibility text

**Optional:**
- **Hashtags**: Relevant hashtags (list separately from caption for easy editing)
- **Text Overlay**: Any text to appear on the image itself
- **Location Tag**: If applicable

## Instagram Carousel

**Required:**
- **Main Caption**: Full post caption
- **Slides**: For each slide (up to 10):
  - **Slide [N] Text**: Complete text for this slide
  - **Slide [N] Image Requirements**: Format (all slides same aspect ratio), subject, style
- **Alt Text**: Descriptive text for the carousel as a whole

**Optional:**
- **Hashtags**: Relevant hashtags
- **Cover Slide Notes**: Special requirements for slide 1 (the preview image)

## Instagram Reel

**Required:**
- **Title**: Reel title
- **Description**: Full description text
- **Content Direction**: What the video should show — scene description, key moments, tone
- **Thumbnail Requirements**: Image specs for the cover frame

**Optional:**
- **Source Timecode**: If cutting from existing video, specify the timecode range
- **Audio**: Music or sound direction
- **Text Overlays**: Any on-screen text with timing
- **Duration**: Target length

## Instagram Story

**Required:**
- **Content Direction**: What each story frame should contain
- **Text Overlays**: On-screen text per frame
- **Image/Video Requirements**: Format (1080x1920), subject, style

**Optional:**
- **Interactive Elements**: Polls, questions, quizzes, links, countdowns
- **Sequence Notes**: If part of a multi-frame story, the order and flow

## Facebook Post

**Required:**
- **Post Text**: Full post copy
- **Image Requirements**: Format (1200x630 recommended), subject, style
- **Alt Text**: Descriptive accessibility text

**Optional:**
- **Link Preview**: If sharing a URL, the expected preview title/description
- **Audience Targeting**: If using targeted posting

## Facebook Reel

**Required:**
- **Title**: Reel title
- **Description**: Full description text
- **Content Direction**: Video content description
- **Thumbnail Requirements**: Cover image specs

**Optional:**
- **Source Timecode**: If cutting from existing video
- **Duration**: Target length

## Facebook Event

**Required:**
- **Event Name**: Title of the event
- **Date & Time**: Start and end times
- **Description**: Full event description
- **Cover Image Requirements**: Format (1200x628), subject, style

**Optional:**
- **Location**: Physical or virtual location details
- **Ticket Link**: URL for registration/tickets
- **Co-hosts**: Other pages or people to list

## YouTube Video (Long-form)

**Required:**
- **Title**: Video title (max 100 characters)
- **Description**: Complete description (first 2-3 lines appear above "Show more")
- **Tags**: Comma-separated tag list
- **Thumbnail Specifications**: Image (1280x720, min width 640px), subject, style
- **Thumbnail Text Overlay**: Any text on the thumbnail

**Optional:**
- **End Screen**: What to promote in the last 20 seconds
- **Cards**: Timed cards linking to other content
- **Chapters**: Timestamp markers with labels
- **Playlist**: Which playlist to add the video to

## YouTube Shorts

**Required:**
- **Title**: Short title
- **Description**: Brief description with relevant hashtags
- **Content Direction**: What the short should show

**Optional:**
- **Source Timecode**: If cutting from longer video
- **Text Overlays**: On-screen text with timing
- **Audio**: Sound/music direction

## LinkedIn Post

**Required:**
- **Post Text**: Full post copy (up to 3,000 characters; first ~140 visible before "see more")
- **Image Requirements**: Format (1200x627 or 1080x1080), subject, style
- **Alt Text**: Descriptive accessibility text

**Optional:**
- **Document/PDF**: If sharing a document carousel, specs for each page
- **Article Link**: If sharing a link, expected preview details
- **Hashtags**: Relevant hashtags (3-5 recommended)

## TikTok

**Required:**
- **Caption**: Post caption (up to 2,200 characters)
- **Content Direction**: Video content description, key moments, tone
- **Hashtags**: Relevant hashtags

**Optional:**
- **Audio**: Sound/music direction
- **Text Overlays**: On-screen text with timing
- **Duration**: Target length
- **Source Timecode**: If cutting from existing video

## Threads

**Required:**
- **Post Text**: Thread text (up to 500 characters per post)
- **Image Requirements**: If visual, format and subject description

**Optional:**
- **Thread Sequence**: If multi-post thread, text for each post in order
- **Alt Text**: Descriptive accessibility text

## Google Ads

**Required:**
- **Ad Group Name**: Name for the ad group
- **Positive Keywords**: Comma-separated keyword list
- **Negative Keywords**: Comma-separated negative keyword list
- **Headlines**: Up to 15 headlines (max 30 characters each)
- **Descriptions**: Up to 4 descriptions (max 90 characters each)
- **Final URL**: Landing page destination

**Optional:**
- **Display Path**: URL path shown in the ad (two 15-character segments)
- **Sitelink Extensions**: Additional links with descriptions
- **Callout Extensions**: Short feature highlights
- **Audience Targeting**: Target audience parameters
- **Budget Notes**: Daily/campaign budget recommendations

## Website Page/Post

**Required:**
- **Page Title**: Title/headline
- **Body Content**: Complete page copy
- **Meta Description**: SEO meta description (max ~160 characters)

**Optional:**
- **Featured Image**: Specifications and alt text
- **URL Slug**: Preferred URL path
- **Categories/Tags**: Content categorization
- **Call to Action**: Primary CTA text and destination
- **SEO Title**: If different from page title (max ~60 characters)
